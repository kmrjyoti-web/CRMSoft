# TravelSis Brand Assets

## Folder Structure

- `design.zip` — Original design file
- `extracted/` — Unzipped contents
- `assets/` — Extracted and processed usable assets
  - `logo.svg` — Brand logo
  - `colors.json` — Brand color palette
  - `screenshots/` — Visual references

## Brand Info

- **Brand Name:** TravelSis
- **Vertical:** TRAVEL
- **Primary Use:** Custom brand skin for Travel vertical

## Integration Path

After extraction:
- Logo → `pc_brand.logo_url`
- Colors → `pc_brand.primary_color`, `secondary_color`
- Custom theme → `pc_brand_vertical_config.custom_theme`
