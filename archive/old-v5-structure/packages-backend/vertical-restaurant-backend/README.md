# @crmsoft/vertical-restaurant-backend

**Status:** PLACEHOLDER

## Purpose

Backend logic specific to the **restaurant** vertical:

- Zod schemas for the `verticalData` JSON column on relevant entities
- Vertical-specific service methods that don't belong in the core API
- Optional cron jobs / scheduled tasks for restaurant-only workflows

## Scope Note

This package is for **CRMSoft customers in the restaurant industry**. It is NOT for CRMSoft's own restaurant-related platform ops (those would live in `apps-backend/api/` directly).
