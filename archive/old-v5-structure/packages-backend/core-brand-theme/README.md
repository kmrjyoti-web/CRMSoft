# @crmsoft/core-brand-theme (backend)

**Status:** PLACEHOLDER

## Purpose

Backend-side brand theme configuration:

- Tenant → brand resolution (which brand a tenant belongs to)
- Brand metadata loading (logo URLs, color palette, fonts, terminology defaults)
- Helpers for the request guard that injects brand context into the request scope

The frontend counterpart lives at `packages-frontend/core-brand-theme/`.
