# @crmsoft/vertical-retail-backend

**Status:** PLACEHOLDER

## Purpose

Backend logic specific to the **retail** vertical:

- Zod schemas for the `verticalData` JSON column on relevant entities
- Vertical-specific service methods that don't belong in the core API
- Optional cron jobs / scheduled tasks for retail-only workflows

## Scope Note

This package is for **CRMSoft customers in the retail industry**. It is NOT for CRMSoft's own retail-related platform ops (those would live in `apps-backend/api/` directly).
