# @crmsoft/vertical-restaurant-frontend

**Status:** PLACEHOLDER

## Purpose

Frontend components specific to the **restaurant** vertical:

- Custom field renderers for the `verticalData` JSON content
- Vertical-specific dashboards and KPI widgets
- Industry-specific form layouts (e.g., for restaurants: menu builder, table layout; for tourism: itinerary builder)

## Scope

For restaurant customers only. Generic CRM UI lives in `packages-frontend/ui-aic/`.
