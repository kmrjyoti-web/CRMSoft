# @crmsoft/core-brand-theme (frontend)

**Status:** PLACEHOLDER

## Purpose

Frontend brand theme:

- CSS variables for brand colors, fonts, spacing
- Theme provider component (`<BrandThemeProvider>`)
- `useBrandTheme()` hook for components to read brand config
- Logo / favicon URL helpers
- Email-template style tokens (for system-generated emails rendered in-app)

The backend counterpart lives at `packages-backend/core-brand-theme/`.
