|--------|------|-------|

| POST | `/api/v1/onboarding/otp/send` | Send email or mobile OTP |

| POST | `/api/v1/onboarding/otp/skip-mobile` | Skip mobile (optional), advance to user_type |
| POST | `/api/v1/onboarding/user-type` | Stage 4: save B2B/B2C/IND_SP/IND_EE |
